Bem Vindo!

Temos um simples objetivo, tentar facilitar nossa vida tento uma api maneira para facilitar a manipulação de imagens em alguns projetos

A ideia e simples e pode ser aplicada por vocês em outros projetos, em python, php, c#, vou dar uma ideia conceitual ou vocês podem manipular essa mesma melhorando e tornando util para vocês

Vamos adicionando alguns recursos simples com o tempo

Redimencionamento

Apatir de um get trazer a imagem ja na dimenção correta que nos precisamos

Marca d'agua

Adicionar um logo pre-selecionado para proteger nossas imagens

Otimização

Vamos tentar reduzir a imagem com algun algoritimos de otimização

Cache

Melhoramento no sistema retorno com cache em memoria
